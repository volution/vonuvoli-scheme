<a id='definition__r7rs__not__description'></a>

#### Description

> ````
> (not obj)
> ````
> 
> 
> The `not` procedure returns `#t` if `obj` is false, and returns
> `#f` otherwise.
> 
> ````
> (not #t)         ===>  #f
> (not 3)          ===>  #f
> (not (list 3))   ===>  #f
> (not #f)         ===>  #t
> (not '())        ===>  #f
> (not (list))     ===>  #f
> (not 'nil)       ===>  #f
> ````
>
