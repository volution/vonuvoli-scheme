<a id='definition__r7rs__length__description'></a>

#### Description

> ````
> (length list)
> ````
> 
> 
> Returns the length of `list`.
> 
> ````
> (length '(a b c))               ===>  3
> (length '(a (b) (c d e)))       ===>  3
> (length '())                    ===>  0
> ````
