<a id='definition__r7rs__boolean_3f__description'></a>

#### Description

> ````
> (boolean? obj)
> ````
> 
> 
> The `boolean?` predicate returns `#t` if `obj` is either `#t` or
> `#f` and returns `#f` otherwise.
> 
> ````
> (boolean? #f)         ===>  #t
> (boolean? 0)          ===>  #f
> (boolean? '())        ===>  #f
> ````
>
