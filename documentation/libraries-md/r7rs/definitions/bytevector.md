<a id='definition__r7rs__bytevector__description'></a>

#### Description

> ````
> (bytevector byte ...)
> ````
> 
> 
> Returns a newly allocated bytevector containing its arguments.
> 
> ````
> (bytevector 1 3 5 1 3 5)         ===>  #u8(1 3 5 1 3 5)
> (bytevector)                     ===>  #u8()
> ````
