<a id='definition__r7rs__vector__description'></a>

#### Description

> ````
> (vector obj ...)
> ````
> 
> 
> Returns a newly allocated vector whose elements contain the given
> arguments.  It is analogous to `list`.
> 
> ````
> (vector 'a 'b 'c)               ===>  #(a b c)
> ````
>
